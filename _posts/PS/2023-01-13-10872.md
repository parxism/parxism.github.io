### 재귀함수 입문.

구조만 파악하면 쉽게 이해할 수 있다.

다만 무한한 반복으로 인해 원하는 값이 출력되지 않을 수 있으니 조건 설정을 잘 해야한다.

어디가 꼬리를 물고 내려가서 다시 올라오는 부분인지 확인해야한다.

```python
import sys
input = sys.stdin.readline

n = int(input())

def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n-1) #여기가 꼬리를 물고 내려가는 부분이다.

print(factorial(n))
```
